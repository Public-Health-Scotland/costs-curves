Next, let’s recode SFR line numbers to map these costs to the correct specialties in the Scottish Morbidity Records(SMR) activity data.

# These are require for matching to SMR01 activity
# combine general surgery (excluding vascular) and vascular surgery line numbers 
# into one "dummy" line number (120)
# combine cardiac surgery and thoracic surgery line numbers into one "dummy" 
# line number (180)
# combine general medicine specialties costed separately into the general 
# medicine line number

sfr_53_recode <- sfr_53 %>% 
  mutate(lineno = case_when(
    #"Clinical Oncology" merged with line 234 "Oncology" in 2019/20 Cost Book  
    lineno == '290' ~ '234',
    # combining general surgery (excluding vascular) and vascular surgery line  
    # numbers into one "dummy" line number (120)
    lineno == '121' ~ '120',
    lineno == '122' ~ '120',
    # combining cardiac surgery and thoracic surgery line numbers into one "dummy" 
    # line number (180)
    lineno == '181' ~ '180',
    lineno == '182' ~ '180',
    # combining general medicine specialties costed separately into the general 
    # medicine line number (230)
    lineno == '231' ~ '230',
    lineno == '345' ~ '230',
    TRUE ~ lineno))

Finally, let’s aggregate the workforce costs at specialty level (ie. aggregate costs by line number). Here we assume that the proportion of the costs that are fixed is 1. We assume this because we have limited data on which proportion of the workforce costs reported are fixed (eg. core staff), and which are variable (eg. NHS staff bank, overtime, etc.).

workforce_costs_by_lineno <- sfr_53 |> 
  group_by(formname, lineno) |> 
  summarise(across(where(is.numeric), ~sum(.x)), .groups = "drop") |> 
  mutate(fixed_costs_prop = 1, variable_costs_prop = 0)
